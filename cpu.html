The Central Processing Unit (CPU) is the brain of a computer. It performs the instructions of a computer program by performing basic arithmetic, logic, control, and input/output operations specified by the instructions. CPUs handle general-purpose computing tasks and are critical to a computer's overall performance.

Key Features of a CPU

  • Clock Speed: Refers to the speed at which the CPU executes instructions, typically measured in GHz (Gigahertz). A higher clock speed means the CPU can perform more operations per second.
  • Core Count: Modern CPUs come with multiple cores, with each core capable of performing tasks independently. More cores allow the CPU to handle multiple tasks simultaneously, improving performance in multi-threaded applications.
  • Threads: Threads are the smallest unit of execution. Some CPUs support multiple threads per core (Hyper-Threading/SMT), which can improve multitasking and performance in certain applications.
  • Cache: Cache is a small amount of very fast memory located inside the CPU to store frequently used data. This helps the CPU avoid waiting for slower main memory, improving overall performance.
  • TDP (Thermal Design Power): TDP is the maximum amount of heat the CPU generates under load, measured in watts. A higher TDP usually indicates more power consumption and heat production, which may require better cooling solutions.
